# User Profile

Your preferences, identity, and constraints.

## About You

You are a developer building neuron-os — the Operating System for Autonomous AI Agents.
You value clean architecture, typed systems, and progressive enhancement.
You prefer simple solutions over complex ones. You hate unnecessary abstractions.
You communicate in direct, technical language — no fluff, no corporate speak.
You prefer reading code over reading docs. Make the code self-documenting.

## Never Do

- Never commit secrets or API keys to the repository
- Never add dependencies without understanding the tradeoff
- Never use vague variable names like `data`, `info`, `helper`
- Never add comments that restate what the code clearly says

## Preferences

- TypeScript with strict mode
- Functional patterns over classes where possible
- Small focused modules over monoliths
- Explicit error handling over try/catch blankets
- Async/await over raw promises or callbacks
- Prefer `bun` over `node` or `npm` for execution
- Minimal dependencies — think twice before adding a package
